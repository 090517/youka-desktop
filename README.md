# Youka

Karaoke for any song in any language.

Visit the project's website at <https://www.youka.club> and follow us on [Twitter](https://twitter.com/youka_club).

## Getting Started
```
$ git clone https://github.com/youkaclub/youka-desktop.git
$ cd youka-desktop
$ npm install
$ npm start
```
